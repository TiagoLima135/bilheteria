Este projeto é um site promocional inspirado no filme **Spider-Man: No Way Home** da Marvel Studios. Ele conta com uma interface visual impactante e interativa, trazendo imagens, links e seções relacionadas ao filme, como **Home**, **Contato**, **Fotos** e **Comentários**. O site foi desenvolvido como parte de um projeto de prática de front-end, explorando habilidades de HTML, CSS.
